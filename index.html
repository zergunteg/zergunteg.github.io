Разработка концепции интерфейса для страницы
оплаты в мобильном приложении IQ Option.

Первый вариант


1. Возможно использование темного фона, при таком подходе поля лучше различимы.

2. Высота полей увеличена и соответствует аналогичным на других экранах. Теперь текст лучше читается, а по полям легче «попасть» пальцем.

3. Лэйблы перенесены налево для экономии места по вертикали.

4. Вместо выпадающего списка с валютами — переключатель. Устраняется лишний «тап».

5. Кнопка оплаты продублирована в «плавающем» хэдере. Теперь она всегда на виду и хорошо визуально соотносится с информацией об оплате.
